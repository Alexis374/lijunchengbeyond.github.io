python regular expression(1)

  • 编译 pattern = re.compile(r'ab+')

  • 多重匹配：re.search查找单个实例，re.findall(pattern,text)返回所有匹配字串的列表，re.finditer(p,t)返回迭代器
  • 模式重复：* + ? {m,n}，表示用在这些字符前面的字符重复的次数。如ab 可以匹配ab?(表示b可以出现一次或者不出现，自然就匹配a了)默认情况下是贪心匹配，在重复指令后加?关闭贪心。ab??表示在可以出现0次或1次b时匹配0次b,同理ab*?也一样
  • 字符集 [ab]表示a或b都可以匹配。[^]用来排除不在字符集内的字符，如[^-. ]表示不是-或.或空格的字符。还有常见的[a-z] [A-Z] [a-zA-Z]的用法
  • 关于贪婪模式的理解：a.*b在字符串中，从左边找a，从右边找b，然后加上中间的所有字串都是匹配结果;非贪婪模式中a.*?b（注意这个?放在表示次数的符号如+ * ?后面或字串的最后，表示非贪婪），从左边寻找a，紧接着向右找到第一个b，连带上中间的所有，即为匹配结果.所以re.findall(r'a.*?b','abbaabbba') 会返回['ab', 'aab']
  • 转义 \d \D \s \S \w(表示字母和数字) \W(非字母数字)
  • 锚定 \A(字符串开始) \Z ^（字符串开始或行开始） $（字符串结束或行结束）
  • \b（单词开头或结尾）\B(不在单词的开头或结尾) （表示前面的一个字符是开始或结束 如 t\b表示以t结束）
    \bt表示以t开头的单词
  • re.match只匹配开头，若不成功则返回空值，re.search可以匹配中间的，但只匹配单个实例
  • 用group解析匹配：小括号分组， 访问单个组匹配的字串，用 match对象的groups()方法，有几个括号则该方法就返回有几个元素的tuple，分别对应该括号里匹配的字符串.命名组：在常规的正则表达式中加入?P<name>如
